## Overview

This repository contains my Week 2 Internship Project focused on analyzing retail sales data and building machine learning models to predict sales.

The project follows the complete data analytics workflow, including data cleaning, exploratory data analysis (EDA), visualization, feature engineering, model building, and performance evaluation.

## Project Tasks

### Task 1: Data Cleaning
- Removed duplicates
- Handled missing values
- Corrected data formats
- Prepared analysis-ready dataset

### Task 2: Exploratory Data Analysis (EDA)
- Distribution Analysis
- Correlation Analysis
- Monthly Sales Trend Analysis
- Customer Purchase Behavior Analysis

### Task 3: Sales Prediction Model
Models Implemented:
- Linear Regression
- Random Forest Regressor

Evaluation Metrics:
- RMSE
- MAE
- R² Score

- Matplotlib
